# Flat mobile menu direction

Diese Iteration reduziert das Interface:

- weniger Glas und weniger Schatten
- flachere Karten und Buttons
- echtes Fraktionslogo aus `public/fraktionslogo.svg`
- mobile Navigation über Hamburger-Menü statt Bottom-Dock
- dezente, flüssige Übergänge

Die Änderung betrifft App-UI, CSS und Dokumentation. Das Logo selbst bleibt unverändert.
