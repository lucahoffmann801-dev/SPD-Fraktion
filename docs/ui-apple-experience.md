# UI Direction

Die Oberfläche orientiert sich an der iOS 26 Design Experience:

- ruhige, großzügige Flächen
- große Hero-Typografie
- transluzente Liquid-Glass-Karten
- weiche, dynamische Controls
- klare mobile Bottom-Navigation
- weniger Verwaltungsgefühl, mehr System-App-Erlebnis

Die Fraktionsfarben bleiben erhalten, werden aber als Akzent genutzt. Der Fokus liegt auf Lesbarkeit, Geschwindigkeit und einem klaren Tages-Cockpit.
