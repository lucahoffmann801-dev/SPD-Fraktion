# Menü-Piktogramme

Bitte die Piktogramme für die Navigation in diesem Ordner ablegen.

Empfohlenes Format:

- SVG bevorzugt
- alternativ PNG oder WebP
- quadratisch, z. B. 512 × 512 px
- transparenter Hintergrund, falls möglich
- einfarbig/dunkelrot passt am besten zum aktuellen Layout

Empfohlene Dateinamen passend zu den Menüpunkten:

- heute.svg
- termine.svg
- aufgaben.svg
- vorgaenge.svg
- ausschuesse.svg
- profile.svg
- fraktion.svg
- dokumente.svg
- kalender.svg

Technischer Pfad in der App:

```text
/menu-icons/termine.svg
```

Wenn die Dateien anders heißen, bitte die Reihenfolge oder Zuordnung kurz mitteilen. Dann können sie in der App sauber gemappt werden.
